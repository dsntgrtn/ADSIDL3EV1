¿QUÉ ES HTML?

El código HTML (HyperText Markup Language), tiene por finalidad el poder de estructurar la página web y desplegar su contenido, para ello este se compone de etiquetas que conforman elementos, de los cuales los más comunes son

  • Encabezados
  • Párrafos
  • Listas
  • Imágenes
  • Vínculos
  • Etc...

QUE SON LAS ESTRUCTURAS DE PROGRAMACIÓN

Las estructuras de programación determinan el orden en que se ejecutan las instrucciones de un algoritmo. Según el teorema de la programación estructurada, cualquier programa puede escribirse combinando tres tipos básicos:

  1. Secuencia Las instrucciones se ejecutan una detrás de la otra, en el mismo orden en que están escritas en el código
  2. Condicionales o Seleccióon: Permiten tomar decisiones y bifurcar el camino del programa dependiendo de si una condición es verdadera o falsa (ej. if, else, switch).
  3. Iteración (Ciclos o Bucles):Permiten repetir un bloque de código mientras se cumpla una condición específica o hasta recorrer un conjunto de datos (ej. for, while, do-while)
